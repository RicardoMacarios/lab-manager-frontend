# 🚰 Sistema de Gerenciamento de Válvulas

Um sistema web desenvolvido para facilitar e organizar a manutenção interna de válvulas industriais. 

Muitas empresas realizam a manutenção de suas próprias válvulas, mas acabam perdendo o controle do histórico, das datas de revisão e do status de cada equipamento. Este projeto nasceu para resolver esse problema, centralizando todas as informações em um painel intuitivo e seguro.

## 🚀 Funcionalidades Atuais
* **Sistema de Autenticação:** Cadastro e Login seguro de usuários.
* **Proteção de Rotas:** Acesso ao Dashboard restrito apenas para funcionários logados.
* **Dashboard Interativo:** Painel inicial com saudação personalizada e botão de logout.


## 🛠️ Tecnologias Utilizadas
Este projeto foi construído utilizando as seguintes tecnologias:
* **Frontend:** HTML5, CSS3 e JavaScript (Vanilla/ES6 Modules).
* **Backend & Banco de Dados:** [Supabase](https://supabase.com/) (Autenticação e Database).
* **Controle de Versão:** Git e GitHub.
